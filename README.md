## Feature Usage Guide

<br/>

**Standard Chat**

Type any question or request into the input bar and press `Enter`. The AI streams its answer in real time. Responses are formatted with WhatsApp-style chat bubbles for clean readability.

<br/>

**Deep Thinking Mode**

For complex logic, mathematical reasoning, or multi-step problem solving:

```
You:   Think deeply about [your complex question]
KWS:   [Displays intermediate reasoning steps live]
KWS:   [Emits final conclusive answer]
```

The Hyper Dual Think Engine activates automatically for sufficiently complex queries, or you can explicitly invoke it by phrasing your request with "think deeply".

<br/>

**Web Search**

Ask naturally for live information:

```
"What is the current Bitcoin price?"
"Show me tutorials on React hooks."
"Latest news on quantum computing."
```

KWS AI routes your query through Google, DuckDuckGo, and Wikipedia simultaneously, then presents an enriched answer with images, video thumbnails, and source links directly in the chat.

<br/>

**Document Upload & RAG Memory**

1. Click the **Paperclip** icon in the input bar
2. Select a PDF, TXT, source code file, or image
3. The AI ingests the file into its Local Vector Memory
4. Ask questions about your document immediately
5. Memory persists at `%APPDATA%/KWS_AI_v9.0_Data` across all future sessions

<br/>

**Voice Input & TTS**

- Click the **Microphone** icon to begin voice input — speak your prompt and KWS AI transcribes it automatically
- Text-to-Speech responses play back automatically when enabled in settings
- Ensure microphone access is granted in `Windows Settings → Privacy & Security → Microphone`

<br/>

**Stop Generation**

Click the **Square (Stop) icon** next to the input bar at any time to immediately halt AI generation or web search.

<br/>

**Internet Learner Agent**

Toggle via Settings or chat command. When active, the agent autonomously crawls reliable knowledge sources in the background and encodes new information into the Vector Database — no user input required.

<br/>

**Managing Session History**

To pin a session: hover over it in the sidebar and click the pin icon.
To lock a session: hover and click the lock icon, then set a password.

## Troubleshooting & Common Issues

<br/>

<details>
<summary><b>"Failed to start: kws_core.bin not found"</b></summary>
<br/>

**Cause:** The core AI model file is missing from the compiled package or `assets/models/` directory.

**Solution:**
- If you downloaded the application, your antivirus may have quarantined `kws_core.bin` — check your antivirus quarantine folder and restore the file
- Re-download and re-extract the application zip archive
- If running from source, ensure `kws_core.bin` is placed in `assets/models/` before launching

</details>

<details>
<summary><b>High Memory / RAM Usage (4 GB – 8 GB)</b></summary>
<br/>

**Cause:** This is expected behaviour. The local AI engine (`kws_core`) loads the entire model directly into your system RAM for fast, offline processing.

**Solution:**
- Close unused browser tabs and background applications to free RAM
- Ensure your system meets the 8 GB RAM minimum requirement
- Consider upgrading to 16 GB RAM for the best experience

</details>

<details>
<summary><b>Application Window Does Not Open</b></summary>
<br/>

**Cause:** KWS AI runs a local Flask server on port 5000. If another application occupies this port, the Eel GUI cannot initialize.

**Solution:**
1. Check if another application (Spotify, another web server, etc.) is using port 5000
2. Verify that Google Chrome or Microsoft Edge is installed — Eel requires a Chromium browser to render the UI
3. Manually open a browser and navigate to `http://127.0.0.1:5000/`

</details>

<details>
<summary><b>Web Search Unresponsive or Timed Out</b></summary>
<br/>

**Solution:**
- Verify your internet connection is active
- If you've made many search requests rapidly, you may have triggered a rate limit with Google or DuckDuckGo — wait 5–10 minutes and retry
- If the issue persists for multiple days, a search provider may have changed their page structure — await a patch update from KWS

</details>

<details>
<summary><b>Chat Database or VectorStore Corrupted</b></summary>
<br/>

**Symptoms:** Past sessions not loading, SQLite database lock errors.

**Solution:**
1. Close KWS AI completely
2. Press `Win + R`, type `%APPDATA%`, press Enter
3. Navigate to `KWS_AI_v9.0_Data`
4. Backup, then delete `chat_history.db` and the `memory_db` folder to fully reset the application

</details>

<details>
<summary><b>Voice Recognition Not Working</b></summary>
<br/>

**Solution:**
1. Navigate to `Windows Settings → Privacy & Security → Microphone`
2. Ensure desktop applications are permitted to access your microphone
3. Verify your default microphone is correctly selected in Windows audio settings

</details>
